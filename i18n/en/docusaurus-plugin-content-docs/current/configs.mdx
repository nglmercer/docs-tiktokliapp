# Configuration extras
Utilities
### Import Configuration:
Import Configurations from a json that are stored in local storage such as:
1. audios
2. data like user, filtered words, names and so on.
3. some previous configurations such as theme, statuses, etc.
### Export Settings: 
Exports the Configurations in a settings.json
 so you can import it with *Import Settings* with all the mentioned data.
